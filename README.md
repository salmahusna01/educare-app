# EduCare App

## Deskripsi
Aplikasi manajemen tugas dan materi pembelajaran

## Fitur
- Login
- Manajemen tugas
- Upload materi
- Monitoring progress

## Tech Stack
- React
- Node.js
- MongoDB

## Cara Menjalankan

### Backend
cd backend  
npm install  
node index.js  

### Frontend
cd frontend  
npm install  
npm start
